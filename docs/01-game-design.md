# RULESHIFT 게임 설계서

## 1. 게임 개요

**제목:** RULESHIFT — 룰을 믿지 마  
**장르:** 60초 생존 아케이드  
**플랫폼:** 웹 브라우저  
**한 줄 소개:** 매 8초마다 바뀌는 규칙과 15초마다 높아지는 스테이지 압력 속에서 오브를 모으고 드론을 피하며 최고 점수를 기록하는 게임.

## 2. 플레이 목표

플레이어는 60초 동안 아레나 안에서 이동하며 오브를 획득합니다. 드론과 위험 구역에 닿으면 점수를 잃고 콤보가 끊깁니다. 룰이 바뀔 때마다 조작 방식, 점수, 적의 행동, 아레나 위험 요소 중 하나가 변합니다.

## 3. 핵심 플레이 루프

1. 오브의 위치를 읽고 이동한다.
2. 현재 룰의 위험 요소를 확인한다.
3. 오브를 먹어 콤보와 점수를 올린다.
4. 8초 뒤 공개될 새 룰과 15초 뒤의 스테이지 상승에 대응한다.
5. 60초 뒤 결과를 확인하고 다시 도전한다.

## 4. 스테이지 시스템

60초 런은 시간 기반의 15초 스테이지 네 개로 나뉩니다. 스테이지는 룰 덱과 독립적으로 결정되므로, 룰의 순서는 매번 바뀌어도 난이도 곡선은 항상 동일합니다. 전환 때는 중앙 배너, 색 플래시, 합성 사운드를 제공하고 플레이어에게 0.9초 무적을 부여합니다. 새 드론은 0.75초 뒤부터 추적해 전환 직후의 즉시 충돌을 막습니다. 점수와 콤보, 현재 룰은 유지됩니다.

| 스테이지 | 시간 | 역할 | 드론 | 속도 | 충돌 페널티 | 오브 수 | 점수 배율 |
| --- | --- | --- | ---: | ---: | ---: | ---: | ---: |
| 1 SYNC | 0–15초 | 학습 구간 | 4 | 0.82× | -60 | 12 | ×1.00 |
| 2 PRESSURE | 15–30초 | 압력 시작 | 5 | 1.00× | -80 | 10 | ×1.25 |
| 3 OVERLOAD | 30–45초 | 과부하 | 6 | 1.16× | -105 | 8 | ×1.55 |
| 4 BREAKPOINT | 45–60초 | 최고 난도 | 7 | 1.32× | -130 | 6 | ×2.00 |

HUD는 `STAGE 01/04`와 현재 이름·배율을 보여주며, 기존 룰 번호는 룰 스트립에서 계속 표시합니다. 오브 획득 점수와 팝업에는 적용된 스테이지 배율이 표시됩니다.

## 5. 규칙 시스템

룰 카드는 게임 시작 시 섞입니다. 최근 3개의 룰 카테고리를 기록해 같은 카테고리가 과도하게 반복되지 않도록 합니다. 완전한 무작위 대신 다음 안전장치를 적용합니다.

- 같은 카테고리 연속 선택 방지
- 초반에는 고난도 룰을 제한
- 동일 룰 재등장 간격 확보
- 룰 변경 직전 화면에 SHIFT IMMINENT 표시
- 룰이 공개된 뒤에는 플레이어가 대응할 수 있도록 유지

현재 룰 카드:

| 룰 | 효과 |
| --- | --- |
| MIRROR MODE | 좌우 입력 반전 |
| OVERDRIVE | 플레이어와 드론 속도 증가 |
| GOLD RUSH | 금빛 오브 점수 3배 |
| ORBITAL DRIFT | 드론이 중앙 주변을 선회 |
| RED ZONE | 아레나 외곽이 위험 구역 |
| LOW SIGNAL | 시야가 어두워짐 |
| BLOOM FIELD | 오브 획득 시 추가 오브 생성 |
| HEAVY HAND | 플레이어 가속과 감속이 무거워짐 |

## 6. 조작과 종료 조건

- 이동: WASD 또는 방향키
- 대시: SPACE
- 일시정지: P 또는 PAUSE 버튼
- 모바일: 화면 방향키와 DASH 버튼
- 사운드: 화면 하단 SOUND 버튼
- 종료: 60초가 지나면 결과 화면 노출

## 7. 디자인 방향

어두운 방송 스튜디오 화면에 산성 라임색 룰 카드와 주황색 경고를 배치합니다. 게임 화면은 짧은 플레이 영상에서도 현재 룰, 남은 시간, 점수 변화가 즉시 보이도록 구성했습니다.

플레이 중에는 다음 피드백을 제공합니다.

- 다음 룰까지 남은 초와 원형 게이지
- 대시 재사용 가능 여부와 쿨다운
- 오브 획득·충돌 시 점수 팝업과 합성 효과음
- 스테이지 전환 배너, 색 플래시, 합성 상승음과 전환 무적
- 플레이 종료 랭크와 브라우저에 저장된 최고 점수
- 탭 전환이나 창 포커스 이탈 시 자동 일시정지

## 8. 사전 과제용 완성 목표

- GitHub Pages에서 링크 클릭 후 바로 플레이
- 30~60초 플레이 영상에서 2개 이상의 룰 변화가 보임
- 외부 API·로그인·유료 라이선스 없이 실행
- 심사자가 1분 안에 목표와 조작을 이해
