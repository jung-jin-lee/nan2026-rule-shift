# RULESHIFT 게임 설계서

## 1. 게임 개요

**제목:** RULESHIFT — 룰을 믿지 마  
**장르:** 60초 생존 아케이드  
**플랫폼:** 웹 브라우저  
**한 줄 소개:** 해금한 스테이지 하나를 선택해 60초 동안 매 8초마다 바뀌는 규칙에 적응하고, 목표 점수로 다음 난이도를 여는 게임.

## 2. 플레이 목표

플레이어는 60초 동안 아레나 안에서 이동하며 오브를 획득합니다. 드론과 위험 구역에 닿으면 점수를 잃고 콤보가 끊깁니다. 룰이 바뀔 때마다 조작 방식, 점수, 적의 행동, 아레나 위험 요소 중 하나가 변합니다.

## 3. 핵심 플레이 루프

1. 오브의 위치를 읽고 이동한다.
2. 현재 룰의 위험 요소를 확인한다.
3. 오브를 먹어 콤보와 점수를 올린다.
4. 8초 뒤 공개될 새 룰에 대응한다.
5. 60초 뒤 목표 달성 여부를 확인하고, 다음 스테이지를 해금하거나 다시 도전한다.

## 4. 스테이지 시스템

스테이지는 시간 구간이 아니라 독립 난이도입니다. 최초에는 STAGE 01만 선택할 수 있습니다. 선택한 스테이지의 설정은 시작부터 정확히 60초 종료까지 고정되며, 점수가 목표 이상이면 다음 스테이지가 영구 해금됩니다. 실패와 재시작은 해금 단계를 되돌리지 않습니다. 준비·종료 화면의 4개 셀렉터는 번호, 난이도 큐, 목표 점수, 해당 스테이지 최고 점수, 잠금 또는 클리어 상태를 보여줍니다. 플레이·일시정지 중에는 선택을 바꿀 수 없습니다.

| 스테이지 | 난이도 큐 | 목표 | 드론 | 속도 | 충돌 페널티 | 오브 수 | 점수 배율 |
| --- | --- | ---: | ---: | ---: | ---: | ---: | ---: |
| 1 SYNC | LEARN THE SIGNAL | 900 | 4 | 0.82× | -60 | 12 | ×1.00 |
| 2 PRESSURE | KEEP YOUR LINE | 1400 | 5 | 1.00× | -80 | 10 | ×1.25 |
| 3 OVERLOAD | MAKE EVERY ORB COUNT | 2000 | 6 | 1.16× | -105 | 8 | ×1.55 |
| 4 BREAKPOINT | HOLD THE IMPOSSIBLE | 2800 | 7 | 1.32× | -130 | 6 | ×2.00 |

HUD는 `STAGE 01/04`, 현재 이름, 점수 배율, 목표 점수를 보여주며, 기존 룰 번호는 룰 스트립에서 계속 표시합니다. 오브 획득 점수와 팝업에는 적용된 스테이지 배율이 표시됩니다. 런 시작 때만 선택 스테이지와 목표를 중앙 배너로 알리고, 런 도중에는 스테이지 전환이 없습니다.

## 5. 규칙 시스템

룰 카드는 게임 시작 시 무작위로 섞입니다. 최근 3개의 룰 카테고리를 기록해 가능한 경우 같은 카테고리가 과도하게 반복되지 않도록 합니다. 다음 안전장치를 적용합니다.

- 섞은 룰 덱을 한 장씩 소진
- 가능한 경우 최근 3개 카테고리와 겹치지 않는 룰 우선 선택
- 룰 변경 직전 화면에 SHIFT IMMINENT 표시
- 공개한 룰을 8초 동안 유지

현재 룰 카드:

| 룰 | 효과 |
| --- | --- |
| MIRROR MODE | 좌우 입력 반전 |
| OVERDRIVE | 플레이어와 드론 속도 증가 |
| GOLD RUSH | 금빛 오브 점수 3배 |
| ORBITAL DRIFT | 드론이 중앙 주변을 선회 |
| RED ZONE | 아레나 외곽이 위험 구역 |
| LOW SIGNAL | 시야가 어두워짐 |
| BLOOM FIELD | 오브 획득 시 추가 오브 생성 |
| HEAVY HAND | 플레이어 가속과 감속이 무거워짐 |

## 6. 조작과 종료 조건

- 이동: WASD 또는 방향키
- 대시: SPACE
- 일시정지: P 또는 PAUSE 버튼
- 모바일: 화면 방향키와 DASH 버튼
- 사운드: 화면 하단 SOUND 버튼
- 종료: 60초가 지나면 FAIL 또는 CLEAR 결과 화면 노출. FAIL은 부족 점수와 재도전을, CLEAR는 즉시 열린 다음 스테이지와 재선택을 안내합니다. STAGE 04를 클리어하면 전체 완료 메시지를 표시합니다.

## 7. 디자인 방향

어두운 방송 스튜디오 화면에 산성 라임색 룰 카드와 주황색 경고를 배치합니다. 게임 화면은 짧은 플레이 영상에서도 현재 룰, 남은 시간, 점수 변화가 즉시 보이도록 구성했습니다.

플레이 중에는 다음 피드백을 제공합니다.

- 다음 룰까지 남은 초와 원형 게이지
- 대시 재사용 가능 여부와 쿨다운
- 오브 획득·충돌 시 점수 팝업과 합성 효과음
- 런 시작 스테이지·목표 배너와 색 플래시
- 플레이 종료 시 FAIL/CLEAR, 목표·부족 점수·다음 스테이지 안내와 스테이지별 최고 점수 저장
- 탭 전환이나 창 포커스 이탈 시 자동 일시정지

## 8. 사전 과제용 완성 목표

- GitHub Pages에서 링크 클릭 후 바로 플레이
- 30~60초 플레이 영상에서 2개 이상의 룰 변화가 보임
- 외부 API·로그인·유료 라이선스 없이 실행
- 심사자가 1분 안에 목표와 조작을 이해
