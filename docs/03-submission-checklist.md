# NAN 2026 제출 체크리스트

## 필수 제출물

- [x] 플레이 가능한 웹 빌드
- [x] 전체 소스 코드
- [x] GitHub Pages 배포 링크
- [ ] 실제 플레이 30~60초 YouTube 영상
- [x] 게임 소개 및 설명 문서 초안
- [x] AI 활용 기술 문서 초안
- [ ] 팀원 롤 기술서 (2인 이상일 때)

## 제출 전 확인

- [x] GitHub 저장소를 public으로 전환하거나 심사 계정 초대
- [x] 모든 커밋 기록 유지
- [x] 링크 클릭 후 별도 설치 없이 플레이
- [ ] PC 실행 파일을 제출하지 않음
- [ ] 영상은 합성 화면이 아닌 실제 플레이 화면으로 촬영
- [ ] 외부 에셋의 출처와 라이선스 명시
- [ ] 제출 링크를 심사 종료 시점까지 유지
- [ ] 마감 전에 제출 내용과 PDF 링크 최종 확인

## 영상 구성 초안

1. 0~05초: 타이틀과 조작을 보여주며 바로 시작
2. 05~20초: 오브 수집과 콤보 상승
3. 20~40초: MIRROR MODE 또는 RED ZONE 룰 변화
4. 40~55초: 대시로 드론을 피하고 점수 폭발
5. 55~60초: 최종 점수와 다시 플레이 버튼
